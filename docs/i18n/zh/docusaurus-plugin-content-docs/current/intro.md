# 简介

**sfork** 是一个 CLI 工具，让你可以在任意对话节点分叉 Claude Code 会话。

## 为什么需要 sfork？

使用 Claude Code 时，你经常会想：

- **探索替代方案** - 从同一起点尝试不同的方法
- **撤销错误** - 回到出错之前重新尝试
- **并行开发** - 同时处理多个功能
- **比较方案** - 用不同方式实现同一功能

sfork 让这一切变得简单——选择对话历史中的任意消息，从该点创建新的会话分支。

## 快速开始

```bash
# 全局安装
npm install -g sfork

# 在任意有 Claude Code 会话的目录运行
sfork
```

## 功能特性

| 功能 | 描述 |
|------|------|
| 可视化历史 | 浏览所有消息及时间戳 |
| 代码变更标记 | 显示哪些消息修改了代码 |
| 灵活过滤 | 显示全部消息或仅用户消息 |
| 终端支持 | 支持 Terminal.app 和 iTerm2 |
